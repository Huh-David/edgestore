# Next.js Basic Access Control Example

This app demonstrates how to use EdgeStore to implement basic access control.
Only signed-in users can access the uploaded files.

## How to run

1. Install dependencies

```bash
npm install
```

2. Add your environment variables to `.env.local`

> See [https://edgestore.dev/docs/quick-start](https://edgestore.dev/docs/quick-start)

3. Run the development server

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) and try to upload something!
